Bienvenue !

Localis is a cartographic online contributive tool. It allows to show on the map points inserted by other users, and add your own.

France map is provided as an example, but an administration interface allows feeding this tool with your own maps in .shp and .dbf format
+ Public workspace on http://developer.berlios.de/projects/localis.
+ Mailing list on localis-general@lists.berlios.de.
+ Permanent meeting place on irc.freenode.net channel #localis.

This project has been leaded in the activities workframe of Média-Cités's Resources Center for Public Multimedia Access Points It's the result of a partnership between Médias-Cité for management ingeneering of data and Makina-Corpus for the tool development.
